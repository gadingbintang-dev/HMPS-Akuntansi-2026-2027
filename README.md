# Website Profil HMPS Akuntansi

Website ini dibuat untuk profil organisasi mahasiswa Himpunan Mahasiswa Program Studi Akuntansi (HMPS Akuntansi) dengan tampilan modern dan responsif dalam struktur multi-page.

## Struktur Direktori

```
HMPS Akuntansi/
├── css/
│   └── styles.css          # File stylesheet utama
├── js/
│   └── script.js           # File JavaScript utama
├── images/
│   ├── logo.png           # Logo organisasi
│   ├── about-image.jpg    # Gambar untuk section tentang kami
│   ├── gallery1.jpg       # Gambar galeri 1
│   ├── gallery2.jpg       # Gambar galeri 2
│   ├── gallery3.jpg       # Gambar galeri 3
│   ├── gallery4.jpg       # Gambar galeri 4
│   ├── gallery5.jpg       # Gambar galeri 5
│   └── gallery6.jpg       # Gambar galeri 6
├── pages/
│   ├── about.html         # Halaman tentang organisasi
│   ├── gallery.html       # Halaman galeri dokumentasi
│   └── contact.html       # Halaman kontak
├── index.html             # Halaman utama/beranda website
├── layout.html            # Template dasar (untuk referensi)
└── README.md              # Dokumentasi ini
```

## Routing/Navigation

- **Beranda**: `index.html`
- **Tentang Kami**: `pages/about.html`
- **Galeri Dokumentasi**: `pages/gallery.html`
- **Kontak Kami**: `pages/contact.html`

## Fitur Utama

1. **Desain Responsif** - Tampil optimal di semua perangkat
2. **Navigasi Mudah** - Dengan menu hamburger untuk mobile
3. **Multi-Page** - Struktur halaman terpisah untuk konten yang lebih terorganisir
4. **Galeri Dokumentasi** - Menampilkan kegiatan organisasi secara lengkap
5. **Kontak Person** - Dengan tombol WhatsApp langsung
6. **Form Kontak** - Untuk pengunjung mengirim pesan
7. **Animasi Halus** - Untuk pengalaman pengguna yang lebih baik

## Cara Menggunakan

1. Simpan semua file dalam struktur direktori seperti di atas
2. Tambahkan file gambar sesuai kebutuhan ke folder `images/`:
   - Logo organisasi (logo.png)
   - Gambar untuk section tentang kami (about-image.jpg)
   - Gambar-gambar dokumentasi (gallery1.jpg - gallery6.jpg)
3. Sesuaikan konten teks dan nomor WhatsApp sesuai kebutuhan
4. Buka index.html di browser untuk melihat hasilnya
5. Gunakan navigasi untuk berpindah antar halaman

## Penyesuaian

Anda dapat menyesuaikan warna, font, dan elemen lain dengan mengedit file `css/styles.css` sesuai kebutuhan organisasi Anda.

## Halaman-Halaman

- **Beranda (index.html)**: Menampilkan preview singkat tentang HMPS Akuntansi
- **Tentang (pages/about.html)**: Informasi lengkap tentang organisasi, visi, misi, dan struktur
- **Galeri (pages/gallery.html)**: Menampilkan dokumentasi kegiatan secara lengkap
- **Kontak (pages/contact.html)**: Informasi kontak person dan form komunikasi
